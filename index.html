How to play:

Streamers:

Just press the Start button and the game will run automatically! You can also choose different question categories by using the dropdown menu.

Viewers:

Once you see a question, type what you think the answer might be in Twitch chat, if you're right, you will get a point. The leaderboard shows the top 5 players.
